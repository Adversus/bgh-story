The scenario manager will allow editing all dynamic facets of an interactive story. These facets are:

  • Stories
    Stories are a collection of scenarios which appear together. They are named for easy idenfication and all scenarios belong to exactly one story.
  • Scenarios
    Each prompt for action is called a scenario. Each scenario any number of possible responses which lead to another scenario.
  • Responses
    A response is a piece of text that is displayed after a given scenario that may be chosen "in response" to the scenario. It points to the next scenario and, optionally, references a fact.
  • Facts
    A fact is an optional piece of a response that is displayed between clicking on a response and the next scenario.
  • Starting and Ending Screens
    Each story has a dedicated piece of text at the start introducing the story and a dedicated piece at the end.
